inheritAttrs：默认情况下父组件在调用子组件时，设置的，不作为props传递的参数，会被应用在子组件的跟节点上，设置为false，可以取消这个默认应用。

为组件绑定v-model，vue在编译后，相当于绑定了value和input两个属性

// v-model
<sInput v-model="sInputData" type="text" />
// 编译后
<sInput :value="sInputData" @input="(value)=>{ sInputData = value }" type="text" />

基于v-model实现input组件和checkbox组件

  • 不同点在于input绑定的值是value，checkbox是checked
  • inheritAttrs和v-bind="$attrs"是为了将父组件设置的非props属性，映射到子组件上

input

// 父组件引用
<sInput v-model="sInputData" type="text" />

// 子组件实现
<template>
  <div>
    <input ref="input" :value="value" @input="onInput" v-bind="$attrs" />
  </div>
</template>

<script>
export default {
  inheritAttrs: false,
  props: {
    value: {
      type: String,
      default: ''
    }
  },
  methods: {
    onInput(e) {
      this.$emit('input', e.target.value)
    }
  }
}
</script>

checkbox

// 父组件引用
<sCheck v-model="sCheckData" type="checkbox" />

// 子组件实现
<template>
  <div>
    <input ref="input" :checked="value" @change="onInput" v-bind="$attrs" />
  </div>
</template>
<script>
export default {
  inheritAttrs: false,
  props: {
    value: {
      type: Boolean,
      default: false
    }
  },
  methods: {
    onInput(e) {
      this.$emit('input', e.target.checked)
    }
  }
}
</script>
