# transform

## rotate 旋转属性

#### rotateX

#### rotateY

#### rotateZ

```css
transform: rotateX(50deg) rotateY(50deg) rotateZ(50deg);
```

```css
transform: rotate(30deg);
```

## skew 倾斜属性 e

#### skewX

#### skewY

```css
向y轴和x轴压缩transform: skew(20deg);
```

## scale 放大属性

```css
放大的倍数transform: scale(2);
```

## translate 变动，位移

```css
xy轴移动transform: translate(120px, 100px);
```

# perspective

```css
/* 父元素的中心点与子元素z轴上的距离
营造一种父元素当前视角的感觉	 */
perspective: 150;
```
