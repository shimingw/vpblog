参考https://medium.com/@auchenberg/live-edit-and-debug-your-react-apps-directly-from-vs-code-without-leaving-the-editor-3da489ed905f

  1. 下载Chrome debugger

  2. 使用create-react-app创建 react 项目

  3. 配置launch.json

    {
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Chrome",
      "type": "chrome",
      "request": "launch",
      "url": "http://localhost:3000",
      "webRoot": "${workspaceRoot}/src"
    }
  ]
}

  4. 执行npm start运行你的项目

  5. 开启vscode的debugger的功能

  6. 打好断点后，修改代码保存或者F5手动刷新debugger，可以触发debuger
