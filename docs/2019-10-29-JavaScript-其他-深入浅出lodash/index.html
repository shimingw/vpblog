# 数组方法

# 拆分方法

  • chunk：将数组（array）拆分成多个 size 长度的区块，并将这些区块组成一个新数组。 如果array 无法被分割成全部等长的区块，那么最后剩余的元素将组成一个区块。

# 链接方法

  • concat：创建一个新数组，将array与任何数组 或 值连接在一起。
  • xor：创建一个给定数组唯一值的数组，使用 symmetric difference（交集取反）做等值比较。返回值的顺序取决于他们数组的出现顺序。
  • xorBy：方法类似 _.xor ，除了它接受 iteratee（迭代器），这个迭代器 调用每一个 arrays（数组）的每一个值，以生成比较的新值。iteratee 调用一个参数： (value).
  • xorWith：该方法是像 _.xor，除了它接受一个 comparator ，以调用比较数组的元素。 comparator 调用 2 个参数：(arrVal, othVal).

# 过滤方法

将某个数组中的值按照特性规则进行过滤

# 根据值过滤

# 纯的函数

  • compact：创建一个新数组，包含原数组中所有的非假值元素。例如false, null, 0, "", undefined, 和 NaN 都是被认为是“假值”。

  • difference：检查某个数组，将特定的值过滤掉，过滤规则为===，如果比较返回false则保留

  • differenceBy：与difference相比，多了一个前置步骤，数组中的每个值在比较前都会过一遍迭代器中的函数，最后比较返回false则保留

  • differenceWith：与difference相比，可以自定义比较方法，不一定使用===进行比较，比较返回false则保留

  • intersection：类似difference，比较规则为===，如果比较返回true则保留（可理解为获取给定数组的交集）

  • intersection：类似differenceBy，比较返回true则保留

  • intersectionWith：类似differenceWith，比较返回true则保留

  • without：创建一个剔除所有给定值的新数组，剔除值的时候使用===

  • filter：类似原生filter

# 不纯的函数

  • pull：移除数组array中所有和给定值相等的元素，使用===

    方法类似_.difference, 不同是这个方法会改变数组 array，并且参数是单个传入

  • pullAll：方法类似_.difference, 不同是这个方法会改变数组 array。

  • pullAllBy：方法类似 _.differenceBy, 不同是这个方法会改变数组 array

  • pullAllWith：方法类似 _.differenceWith , 不同是这个方法会改变数组 array

  • remove：根据规则，移除array中对应的元素，并返回被移除元素的数组。

# 根据索引过滤

# 不纯的函数
  • pullAt：根据索引 indexes，移除array中对应的元素，并返回被移除元素的数组。

# 切片方法

获取某个数组中，特定的值

  • drop：创建一个切片数组，去除array前面的n个元素。（n默认值为 1。）
  • dropRight：创建一个切片数组，去除array尾部的n个元素。（n默认值为 1。）
  • dropRightWhile：从左往右截取数组，直到返回 false，停止截取
  • dropWhile：从左往右截取数组，直到返回 false，开始截取

# 填充方法

  • fill：使用 value 值来填充（替换） array，从start位置开始, 到end位置结束（但不包含end位置）。

    Note: 这个方法会改变 array（不是创建新数组）。

# 查找方法

根据规则，获取数组中，某个位置的值或索引

# 根据值获取索引

  • findIndex：比原生findIndex，多了一个参数，可以指定查找的起始索引

  • findLastIndex：与findIndex类似，区别是它是从右到左进行查找，同时也可以指定查找的起始索引

  • indexOf：与原生的indexof一致，功能类似findIndex

  • lastIndexOf：类似indexOf，从右到左查找

  • includes：检索某个值是否在数组中，同时也可以用来检索对象的 key，或者字符串的include

# 根据索引获取值

  • head：获取数组 array 的第一个元素（ps：参数如果不为数组则返回undefined，优于arr[0]）
  • initial：获取数组array中除了最后一个元素之外的所有元素（去除数组array中的最后一个元素）
  • last：获取array中的最后一个元素。好处同head
  • nth：获取array数组的第 n 个元素。如果n为负数，则返回从数组结尾开始的第 n 个元素。
  • slice：裁剪数组array，从 start 位置开始到end结束，但不包括 end 本身的位置。
  • sortedIndex：用二进制的方式进行比较，获取某个值插入数组尽可能小的索引位置
  • sortedIndexBy：与sortedIndex类似，区别是，在比较前，可使数组中的值经过特定方法的转换
  • sortedIndexOf：这个方法类似 _.indexOf，它只能在已排序的数组上进行检索
  • sortedLastIndex：此方法类似于 _.sortedIndex，检索尽可能大的索引
  • sortedLastIndexBy：与sortedIndexBy类似，获取尽可能大的索引
  • sortedLastIndexOf：与sortedIndexOf，类似获取尽可能大的索引
  • tail：获取除了array数组第一个元素以外的全部元素。
  • take：创建一个数组切片，从array数组的起始元素开始提取n个元素。
  • takeRight：创建一个数组切片，从array数组的最后一个元素开始提取n个元素。
  • takeRightWhile：与takeRight类似，区别是直到返回false，停止。
  • takeWhile：与takeRightWhile类似，区别是从头开始取值。

# 根据规则获取值

  • find：类似原生findOne
  • findLast：区别于find是从后往前找

# 去重方法

  • sortedUniq：这个方法类似 _.uniq，但是它只会对相邻的值进行去重
  • sortedUniqBy：这个方法类似 _.uniqBy，但是它只会对相邻的值进行去重
  • union：对多个数组进行去重，比较规则===
  • unionBy：方法类似 _.union，在每个值比较前，可对每个值进行一次转换
  • unionWith：方法类似 _.union，在两个值比较时，可以自定义比较规则
  • uniq：对单个数组中的值进行去重，比较规则===
  • uniqBy：这个方法类似 _.uniq ，在每个值比较前，可对每个值进行一次转换
  • uniqWith：这个方法类似 _.uniq，在两个值比较时，可以自定义比较规则

# 减少嵌套方法

  • flatten：减少一级array嵌套深度。
  • flattenDeep：将array递归为一维数组。
  • flattenDepth：根据 depth 递归减少 array 的嵌套层级

# 数组转换方法

将数组以对象或者字符串的数据结构展现出来

  • join：将 array 中的所有元素转换为由 separator 分隔的字符串。

  • fromPairs：根据数组中的值返回一个由键值对pairs构成的对象

  • zipObject：这个方法类似 _.fromPairs，除了它接受 2 个数组，第一个数组中的值作为属性标识符（属性名），第二个数组中的值作为相应的属性值。

  • zipObjectDeep：这个方法类似 _.zipObject，除了它支持属性路径。

  • countBy：创建一个组成对象，key是经过迭代器后返回的值，value这个值返回的次数

  • groupBy：类似countBy，不同是value用来体现匹配中的集合。

# 排序方法

# 不纯的函数

  • reverse：反转数组

# 数组分组

  • unzip：它接收分组元素的数组，并且创建一个数组
  • unzipWith：此方法类似于 _.unzip，除了它接受一个iteratee指定重组值应该如何被组合
  • zip：创建一个分组元素的数组，数组的第一个元素包含所有给定数组的第一个元素，数组的第二个元素包含所有给定数组的第二个元素，以此类推。
  • zipWith：这个方法类似于 _.zip，不同之处在于它接受一个 iteratee（迭代函数），来 指定分组的值应该如何被组合。 该 iteratee 调用每个组的元素： (...group).

# 数组判断方法

  • every：判断数组中的所有值是否都满足某个条件，否则返回false，是则返回true

# 遍历方法

  • forEach：类似原生的forEach，区别是可以遍历对象

  • forEachRight：这个方法类似 _.forEach，不同之处在于， _.forEachRight 是从右到左遍历集合中每一个元素的。

  • flatMap：遍历数组，将创建的值，进行扁平化合并

  • flatMapDeep：这个方法类似 _.flatMap 不同之处在于， _.flatMapDeep 会继续扁平化递归映射的结果。

  • flatMapDepth：该方法类似 _.flatMap，不同之处在于， _.flatMapDepth 会根据指定的 depth（递归深度）继续扁平化递归映射结果。

  • map：类似原生map，不同可遍历对象，根据对象的value，生成新的数组

# 分类方法

  • partition：可根据规则对数组中的值进行分类，只能分 2 类

# 函数工具方法

  • after：当某个函数被触发指定次数后发出，某个回调
  • ary：创建一个调用func的函数。调用func时最多接受 n个参数，忽略多出的参数。
  • before：限制某个函数的调用次数
  • curry：使某个可以接受多个参数的函数柯理化
  • curryRight：与curry类似，从参数的右侧开始柯理化
  • debounce：防抖，函数在延迟设定时间后执行，如果，在此期间又触发了函数，则继续顺延
  • defer、delay：类似setTimeout
  • negate：使一个函数结果的返回值取反
  • once：使一个函数只能执行一次，重复调用不执行函数，直接返回第一次的结果
  • overArgs：创建一个函数，调用func时参数为相对应的transforms的返回值。如果没有对应的转换函数则不进行转换
  • spread：func传入的数组可以解析成单个的值
  • throttle：节流，在设定时间内，只能执行一次函数。

# 数据工具方法

  • clone：创建一个 value 的浅拷贝
  • cloneDeep：创建一个 value 的深拷贝
  • cloneDeepWith：类似cloneDeep，区别，在拷贝时，可以return指定值作为返回值
  • cloneWith：类似clone，区别，在拷贝时，可以return指定值作为返回值
  • conformsTo：判断对象中某个 key 的 value 是否满足判断条件
  • isEqual：执行递归比较

# 对象方法

  • forIn：遍历对象，会遍历原型链上可枚举的值
  • forOwn：遍历对象，不会遍历原型链上可枚举的值
  • pick：创建一个从 object 中选中的属性的对象。
  • pickBy：类似pick，可在创建时，自定义规则进行选择
  • merge：合并两个对象，类似webpack-merge效果
  • get：对象取值方法，可设置默认值
  • result：与get类似，如果是函数，则返回函数执行后的结果
