# 户型选择

```
1、客厅宽度超过三米的墙，放沙发和电视机
2、大门不能对着卧室，保护隐私
```

```

	1、有玄关
	2、客厅卧室朝南
	3、阳台在客厅

# 何时买房最好

```

	15.3

	11.2
	14.4
	17年初

预计2018年初是一个抄底的好时机
2020年初是一个下一个高点

南京库存5W-5.5W是库存最大的时期

1、库存确认回落的第一个月或者第二个月
2、开一栋楼区划67成

库存高位回落的第二个月月底+涨幅10%以内
库存高位回落+涨幅10% = 买
